# FishMol
A python molecular dynamics data analysis tool.

## Install
### Requirements
- python3: >=3.8

### Install
The easiest way is to install by `pip`
```bash
pip install fishmol
```
Run the following command in your terminal to see if fishmol is successfully installed:
```bash
fishmol
```
